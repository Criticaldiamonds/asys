Help using Asys

General FAQ:

- Q: How do I close tabs?
- A: Right-click in the document, and select any of the three 'Close' options from the context menu.

- Q: How do I add images to my document?
- A: Use the Insert->Image menu item to add images to the current document.
  (Note that not all file types support images)

- Q: What do the numbers in the bottom left mean?
- A: In order, they are: Total Characters, Total Words, Current Column, Current line

- Q: How do I change which toolbars are shown?
- A: The View menu contains visibility options. These options are saved.

- Q: How do I disable developer message completely?
- A: You can currently only disable the current message from showing.
  The dialog will show again if the message is changed.
